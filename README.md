# ANOVIC Growth OS

From Lead to Growth.

ANOVIC Growth OS is a business intelligence and sales automation platform built to help marketing agencies analyze leads, audit digital presence, score opportunities, generate growth reports, and create sales assets faster.

## Vision

Turn any business lead into a complete growth audit, strategy report, and sales-ready proposal in less than 30 minutes.

## Core Modules

- Lead Intelligence
- Digital Audit Engine
- Scoring Engine
- Competitor Intelligence
- Growth Intelligence
- Report Generator
- Sales Assets Generator
- Knowledge Base

## MVP Scope

The first version will support:

- Add and manage leads
- Run digital audits
- Calculate scores
- Generate recommendations
- Create executive summaries
- Generate WhatsApp, email, and call scripts
- Export reports as PDF/PPTX

## Tech Stack

- Frontend: Next.js + TypeScript
- Backend: FastAPI
- Database: PostgreSQL
- Auth: Supabase Auth
- Storage: Supabase Storage
- AI: OpenAI API
- Deployment: Vercel + Railway
